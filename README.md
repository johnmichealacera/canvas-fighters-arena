# Canvas Fighters Arena

A fast-paced 2D fighting game built with Next.js and HTML5 Canvas.

## Features

- Full-screen canvas with smooth `requestAnimationFrame` game loop
- Character select with multiple fighters and custom sprite packs
- Per-character animation setup (sheet, sequence, atlas support)
- Per-character sizing and offsets for visual tuning
- Physics with gravity and jumping
- Hit detection, health system, KO/TIME UP flow, and round tracking
- Modern HUD with character names, health bars, and countdown timer

## Controls

### Player 1

- Move: `W`, `A`, `D`
- Attack: `F`

### Player 2

- Move: `Arrow Up`, `Arrow Left`, `Arrow Right`
- Attack: `L`

## Scripts

- `npm run dev` - Start local development server
- `npm run build` - Build production output
- `npm run lint` - Run Next.js lint checks

## Run Locally

1. Install dependencies:

   `npm install`

2. Start the game:

   `npm run dev`

3. Open:

   `http://localhost:3000`
